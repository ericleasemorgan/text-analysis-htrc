<h1>Initialize HathiTrust collection</h1>
<p>Given an rsync file created by an algorithm of the HathiTrust Research Center, this form allows you to create a local cache of HathiTrust content, have some rudimentary analysis done against it, and read the the corpus "distantly". If you do not have an rsync file, then visit the <a href="https://sharc.hathitrust.org">Center</a> to create your own, or you can use any of the following files as sample data: <a href="../etc/rsync-cookery.sh">cookery</a>, <a href="../etc/rsync-lancaster.sh">Lancaster</a>, <a href="../etc/rsync-yeats.sh">Yeats</a>.</p>
